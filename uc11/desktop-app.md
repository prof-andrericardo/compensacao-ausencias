# 🎨 PROJETO: DESIGN E PROTÓTIPO DO APP

## Sua Missão
Criar um app desktop para ajudar estudantes! Algumas ideias:
- Organizador de Estudos
- Agenda de Provas
- Controle de Notas
- Biblioteca Virtual
- Sua ideia criativa!

### FASE 1: Design no Figma (25 pontos)

**Telas Obrigatórias:**
1. **Tela de Login**
   - Campo de usuário
   - Campo de senha
   - Botão de entrar
   - Opção "esqueci a senha"

2. **Tela Principal**
   - Menu de navegação
   - Área de conteúdo
   - Barra de status
   - Perfil do usuário

3. **Telas Funcionais**
   - Lista de itens
   - Formulário de cadastro
   - Visualização de dados
   - Configurações

**Requisitos de Design:**
- Esquema de cores consistente
- Ícones intuitivos
- Tipografia adequada
- Design responsivo

### FASE 2: Protótipo Interativo (25 pontos)

1. **Navegação**
   - Links entre telas
   - Animações simples
   - Menus funcionais
   - Botões interativos

2. **Experiência do Usuário**
   - Feedback visual
   - Mensagens de erro
   - Confirmações
   - Dicas de uso

## Entregáveis

1. **Arquivo do Figma**
   - Todas as telas
   - Components organizados
   - Protótipo navegável
   - Style guide

2. **Documentação**
   - Screenshots das telas
   - Explicação das funcionalidades
   - Fluxo de navegação
   - Guia de estilos

## ⭐ Pontos Extras (10 pontos)
- Modo escuro
- Animações personalizadas
- Micro-interações
- Versão mobile

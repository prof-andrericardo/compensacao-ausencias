# 📚 Atividades de Compensação de Ausências - Curso Técnico em Informática

## 📂 UC7 - Sistemas Operacionais de Redes

### 🎯 Objetivo
Introdução aos conceitos de Sistemas Operacionais de Rede usando Linux (Debian) como plataforma de aprendizado.

### 📑 Conteúdo Programático
1. Linux Básico
   - Introdução ao Linux
   - Distribuição Debian
   - Comandos básicos
   - Administração básica

2. Serviços de Rede
   - Servidor DNS (bind9)
   - Servidor Web (nginx)
   - Gerenciamento de serviços
   - Segurança básica

### 💻 Ferramentas Necessárias
- Hyper-V
- Debian Linux (última versão estável)
- Acesso à internet para downloads

### 📝 Trabalho de Compensação

#### Parte Teórica (40% da nota)
**Tema: "Entendendo Servidores Linux"**
1. Linux Básico (20 pontos)
   - Conceitos fundamentais
   - Sistema Debian
   - Diferenças servidor/desktop
   - Comandos essenciais

2. Serviços de Rede (20 pontos)
   - Fundamentos DNS
   - Servidor Web básico
   - Gerenciamento de usuários
   - Segurança essencial

#### Parte Prática (60% da nota)
**Projeto: Servidor Web Básico**

1. Instalação (20 pontos)
   - Configuração da VM
   - Instalação Debian
   - Configuração de rede
   - Atualizações do sistema

2. Serviços (40 pontos)
   - Instalação/configuração DNS
   - Instalação/configuração nginx
   - Página web básica
   - Testes de funcionamento

### ⭐ Pontos Extras (10 pontos)

# 🎵 Projeto: Escritório TikTok Brasil

## Sua Missão
Montar a rede do escritório no Packet Tracer, com todas as áreas importantes!

### 1️⃣ Áreas para Criar (20 pontos)

**Área dos Criadores**
- 5 computadores para edição
- Rede rápida para upload de vídeos
- WiFi para convidados

**Área da Moderação**
- 3 computadores
- Acesso aos servidores
- Rede separada e segura

**Área Administrativa**
- 4 computadores
- 1 impressora
- Rede própria

### 2️⃣ Configurações Legais (20 pontos)

**Tarefas:**
1. Redes Separadas
   - Cada área com seu próprio IP
   - WiFi para visitantes
   - Senhas diferentes para cada rede

2. Serviços Básicos
   - DHCP funcionando
   - Compartilhar arquivos
   - Acesso à impressora

### 3️⃣ Testes Finais (10 pontos)
- Verificar se tudo se conecta
- Testar o WiFi
- Confirmar se a impressora funciona
- Ver se os arquivos são compartilhados

## 🎯 O Que Entregar

1. **Arquivo do Packet Tracer**
   - Salvar como "TikTok_SeuNome.pkt"
   - Organizar os equipamentos na tela

2. **Documento Explicando**
   - Como você organizou tudo
   - Que equipamentos usou
   - Os IPs de cada rede
   - Prints mostrando que funciona

## ⭐ Pontos Extras! (10 pontos)
- Criar página web do TikTok
- Colocar logo nas telas
- Fazer backup automático
- Ideias criativas suas!


**DICAS IMPORTANTES 🚀**

📅 **Quando Entregar:**
- Parte 1: Em 15 dias
- Parte 2: Em 30 dias

📝 **Como Fazer Bonito:**
1. Comece desenhando no papel
2. Faça um pouco cada dia
3. Tire prints de tudo que fizer
4. Peça ajuda se precisar!

🎮 **Ferramentas Permitidas:**
- Packet Tracer
- Internet para pesquisar
- PowerPoint ou Word
- Paint ou ferramentas de desenho

🆘 **Precisa de Ajuda?**
- Pode mandar email com dúvidas
- Perguntar durante as aulas
- Consultar material antigo
- Pesquisar na internet

💡 **Dicas de Ouro:**
1. Salve seu trabalho sempre!
2. Faça backup em pen drive
3. Organize bem os arquivos
4. Teste tudo antes de entregar

**LEMBRE-SE:**
- Não existe dúvida boba

- Use sua imaginação

## Requisitos do Projeto

### FASE 1: Preparação dos Dados (20 pontos)

1. **Organização da Planilha**
```
- Coletar dados (mínimo 30 itens)
- Organizar em colunas
- Limpar dados inconsistentes
- Formatar corretamente
```

2. **Estrutura Necessária**
```
- Título/Nome
- Dados numéricos (3+ colunas)
- Dados categóricos (2+ colunas)
- Datas (se aplicável)
```

### FASE 2: Análise dos Dados (20 pontos)

1. **Cálculos Básicos**
```excel
- Média, Mediana, Moda
- Máximos e Mínimos
- Contagens
- Porcentagens
```

2. **Análises Específicas**
```excel
- Tendências
- Comparações
- Rankings
- Distribuições
```

### FASE 3: Visualização (20 pontos)

1. **Gráficos Obrigatórios**
```excel
- Gráfico de barras/colunas
- Gráfico de pizza
- Gráfico de linha temporal
- Gráfico combinado
```

2. **Dashboard Final**
```excel
- Resumo visual dos dados
- Principais indicadores
- Filtros interativos
- Layout organizado
```

## Requisitos do Projeto

### 1. HTML Básico (10 pontos)
- Título do jogo
- Área do jogo
- Placar
- Botões de controle

### 2. CSS Básico (10 pontos)
- Cores legais
- Layout organizado
- Design responsivo
- Animações simples

### 3. JavaScript (30 pontos)
- Variáveis organizadas
- Funções que funcionem
- Eventos de clique
- Lógica do jogo
- Sistema de pontos

### 4. Extras (10 pontos bônus)
- Sons no jogo
- Ranking de jogadores
- Níveis de dificuldade
- Tela de início

# 🏫 Projeto: Criando a Rede da Escola Tech

## Cenário
Parabéns! Você foi escolhido(a) para criar a rede de uma escola de tecnologia.

## O que você precisa fazer:

### 1️⃣ Primeira Etapa: Planejamento (10 pontos)
- Criar um desenho simples da rede
- Definir quais equipamentos vai usar
- Escolher os IPs para cada sala

### 2️⃣ Segunda Etapa: Montagem da Rede (20 pontos)

**Laboratório de Informática**
- 1 switch
- 10 computadores
- Todos devem pegar IP automaticamente (DHCP)
- Rede: 192.168.10.0/24

**Sala dos Professores**
- 1 switch
- 5 computadores
- Rede: 192.168.20.0/24
- 1 impressora de rede

**Conectando Tudo**
- 1 roteador principal
- WiFi para os alunos
- Servidor DHCP funcionando

### 3️⃣ Terceira Etapa: Testes (20 pontos)
- Fazer todos os computadores se comunicarem
- Testar se o DHCP está funcionando
- Verificar se a impressora está acessível
- Testar o WiFi

## Como Entregar
1. Arquivo do Packet Tracer (.pkt)
2. Documento com:
   - Print das telas mostrando a rede funcionando
   - Explicação do que você fez
   - Lista de problemas que encontrou e como resolveu

## Dicas de Ouro 🌟
- Use o roteador 2911
- Use switches 2960
- Configure nomes em todos os equipamentos
- Teste tudo usando o comando ping
- Organize os equipamentos na tela
- Capriche na documentação

## Bônus! ⭐ (10 pontos extras)
- Adicionar servidor web com página da escola
- Criar VLANs para separar as redes
- Configurar senhas nos equipamentos

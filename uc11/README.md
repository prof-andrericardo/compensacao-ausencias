# 🎨 PROJETO: DESIGN E PROTÓTIPO DO APP

## Sua Missão
Criar um app desktop para ajudar estudantes! Algumas ideias:
- Organizador de Estudos
- Agenda de Provas
- Controle de Notas
- Biblioteca Virtual
- Sua ideia criativa!

### FASE 1: Design no Figma (25 pontos)

**Telas Obrigatórias:**
1. **Tela de Login**
   - Campo de usuário
   - Campo de senha
   - Botão de entrar
   - Opção "esqueci a senha"

2. **Tela Principal**
   - Menu de navegação
   - Área de conteúdo
   - Barra de status
   - Perfil do usuário

3. **Telas Funcionais**
   - Lista de itens
   - Formulário de cadastro
   - Visualização de dados
   - Configurações

**Requisitos de Design:**
- Esquema de cores consistente
- Ícones intuitivos
- Tipografia adequada
- Design responsivo

### FASE 2: Protótipo Interativo (25 pontos)

1. **Navegação**
   - Links entre telas
   - Animações simples
   - Menus funcionais
   - Botões interativos

2. **Experiência do Usuário**
   - Feedback visual
   - Mensagens de erro
   - Confirmações
   - Dicas de uso

## Entregáveis

1. **Arquivo do Figma**
   - Todas as telas
   - Components organizados
   - Protótipo navegável
   - Style guide

2. **Documentação**
   - Screenshots das telas
   - Explicação das funcionalidades
   - Fluxo de navegação
   - Guia de estilos

## ⭐ Pontos Extras (10 pontos)
- Modo escuro
- Animações personalizadas
- Micro-interações
- Versão mobile


**PARTE 2 - APRESENTAÇÃO DO PROJETO (50%)**

```markdown
Criar uma apresentação criativa com:

   - Público-alvo

   - Funcionalidades
   - Próximos passos
```

**DICAS PARA ARRASAR! 🚀**

⏰ **Prazos:**
- Design: 15 dias
- Protótipo: 15 dias

💻 **Ferramentas:**
- Figma (design)
- VSCode (documentação)
- Navegador para pesquisa

🎯 **Dicas de Ouro:**
1. Pesquise apps similares
2. Faça rascunhos no papel
3. Use componentes no Figma
4. Teste com amigos

📚 **Inspiração:**
- Behance
- Dribbble
- Pinterest
- Instagram UI

**PASSO A PASSO:**

1. **Ideação**
   - Defina o problema
   - Pesquise soluções
   - Liste funcionalidades
   - Faça rascunhos

2. **Design**
   - Crie style guide
   - Design das telas
   - Teste usabilidade
   - Refine o visual

3. **Protótipo**
   - Crie interações
   - Teste navegação
   - Ajuste problemas
   - Prepare apresentação

**LEMBRE-SE:**
- Design é iterativo
- Simplicidade é melhor
- Teste bastante
- Peça feedback

💡 **Dicas de Design:**
- Use cores que combinam
- Mantenha consistência
- Pense no usuário
- Menos é mais

**AVALIAÇÃO:**
- Criatividade
- Usabilidade
- Organização
- Apresentação

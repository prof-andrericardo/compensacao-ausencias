# 🎮 PARTE 2 - MISSÃO PRÁTICA: PROTEGER O CAMPEONATO! (50%)

## Sua Missão
O maior campeonato de e-sports do Brasil precisa da sua ajuda para ficar seguro!

### FASE 1: Base Segura no Windows Server (25 pontos)

**Missão 1: Fortaleza Digital**
1. **Criar o QG**
   - Montar servidor seguro
   - Criar senhas super fortes
   - Ativar escudo (firewall)
   - Instalar proteções

2. **Organizar as Tropas**
   - Criar grupos:
     * Pro Players
     * Narradores
     * Staff
   - Definir quem pode fazer o quê
   - Proteger as estratégias secretas

3. **Sistema de Backup**
   - Salvar tudo importante
   - Testar recuperação
   - Fazer backup automático

### FASE 2: Rede Protegida no Packet Tracer (25 pontos)

**Missão 2: Defesa em Camadas**
1. **Montar o Campo de Batalha**
   - Separar as redes:
     * Área dos jogadores
     * Área da transmissão
     * WiFi do público
   - Criar zonas seguras

2. **Ativar as Proteções**
   - Colocar senhas fortes
   - Configurar firewall
   - Criar regras de acesso
   - Monitorar tudo

## O Que Você Precisa Entregar

1. **Seus Arquivos**
   - Backup do servidor
   - Arquivo do Packet Tracer
   - Manual de proteção
   - Prints das configurações

2. **Relatório de Missão**
   - Fotos do que fez
   - Lista de proteções
   - Problemas que achou
   - Como resolveu tudo

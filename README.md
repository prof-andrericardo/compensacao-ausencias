## 💻 Como Usar Este Repositório

1. Cada pasta de UC contém:
   - README específico com instruções
   - Material teórico
   - Atividades práticas
   - Recursos necessários

2. Para começar:
   ```bash
   git clone https://github.com/seu-usuario/nome-do-repo
   cd nome-do-repo
   ```

## 📝 Instruções Gerais

1. **Prazos:**
   - Parte Teórica: 15 dias
   - Parte Prática: 30 dias

2. **Entrega:**
   - Teoria: Formato PDF/PPT
   - Prática: Arquivos específicos de cada UC

3. **Avaliação:**
   - Teoria: 40% da nota
   - Prática: 60% da nota
   - Pontos extras disponíveis

# UC7 - Sistemas Operacionais de Redes

## 📑 Descrição
Unidade Curricular focada em Sistemas Operacionais de Rede usando Linux (Debian) como plataforma de aprendizado.

## 🎯 Trabalho de Compensação de Ausências

### Tema: "Meu Primeiro Servidor Linux"

### 📚 PARTE 1 - TEORIA (40% da nota)
**Missão: "Entendendo Servidores Linux"**

Criar uma apresentação explicando:
1. Linux Básico (20 pontos)
   - O que é Linux?
   - Por que usar Debian?
   - Diferença entre desktop e servidor
   - Comandos básicos do Linux

2. Serviços de Rede (20 pontos)
   - O que é DNS e para que serve
   - Como funciona um servidor web
   - Usuários e permissões no Linux
   - Importância da segurança básica

### 💻 PARTE 2 - PRÁTICA (60% da nota)

#### Ferramentas Utilizadas:
- Hyper-V para virtualização
- Debian Linux (última versão estável)

#### Projeto: Servidor Web Básico

1. **Instalação do Sistema (20 pontos)**
   - Criar máquina virtual no Hyper-V
   - Instalar Debian (modo servidor)
   - Configurar rede básica
   - Atualizar o sistema

2. **Configuração de Serviços (40 pontos)**
   - Instalar e configurar DNS (bind9)
   - Instalar e configurar Web (nginx)
   - Criar página web simples
   - Testar os serviços

### ⭐ Pontos Extras (10 pontos)
- Página web personalizada
- Configuração de hosts virtuais no nginx
- Documentação com screenshots
- Tutorial passo a passo

## 📅 Prazos
- Teoria: 15 dias
- Prática: 30 dias

## 📝 Requisitos Mínimos
- Computador com Hyper-V
- 2GB RAM livre
- 20GB de espaço em disco
- Acesso à internet

## 🔍 Critérios de Avaliação
1. Teoria (40%)
   - Clareza nas explicações
   - Exemplos práticos
   - Organização do conteúdo

2. Prática (60%)
   - Sistema instalado corretamente
   - Serviços funcionando
   - Testes realizados

## 💡 Dicas Importantes
- Faça backups da VM
- Anote todos os comandos usados
- Tire screenshots dos processos
- Teste cada serviço após instalar

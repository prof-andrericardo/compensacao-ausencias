# 📂 UC9 - Projeto Integrador

### 🎯 Objetivo
Desenvolver um projeto prático que integre os conhecimentos adquiridos nas outras UCs do curso, especialmente:
- Redes de Computadores
- Sistemas Operacionais
- Desenvolvimento de Software
- Design Web
- Segurança

### 💡 Proposta de Projeto: "Sistema Web com Infraestrutura Completa"

#### 📝 Requisitos do Projeto

1. **Desenvolvimento Web (30%)**
   - Frontend em HTML, CSS e JavaScript
   - Design responsivo
   - Interface intuitiva
   - Formulários e validações

2. **Infraestrutura de Rede (30%)**
   - Servidor Linux (Debian)
   - Configuração DNS
   - Servidor Web (nginx)
   - Documentação de rede

3. **Segurança (20%)**
   - HTTPS
   - Senhas seguras
   - Backup dos dados
   - Logs de acesso

4. **Documentação (20%)**
   - Manual do usuário
   - Documentação técnica
   - Diagramas e fluxos
   - Apresentação final

### 💻 Ferramentas Necessárias
- VSCode (desenvolvimento)
- Hyper-V (virtualização)
- Pacote Office (documentação)
- Navegador moderno

   - Requisitos

2. **Desenvolvimento**
   - Frontend

   - Infraestrutura
   - Testes

3. **Implementação**
   - Deploy

4. **Apresentação**
   - Preparação

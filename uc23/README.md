# UC23 - Internet das Coisas (IoT)

## 🎯 Trabalho de Compensação de Ausências

### 📱 PARTE 1: TEORIA DA IoT (40%)

**Missão: "Explorando o Mundo Conectado"**
Criar uma apresentação sobre:

1. **Fundamentos de IoT (20 pontos)**
   - O que é IoT?
   - Como funciona?
   - Principais tecnologias
   - Aplicações no dia a dia
   Use exemplos como:
   - Smart Home
   - Wearables
   - Cidades Inteligentes
   - Indústria 4.0

2. **Segurança e Privacidade (20 pontos)**
   - Riscos em IoT
   - Proteção de dados
   - Melhores práticas
   - Casos reais
   Exemplos:
   - Câmeras invadidas
   - Vazamento de dados
   - Ataques a dispositivos
   - Soluções de segurança

### 💻 PARTE 2: PROJETO PRÁTICO NO PACKET TRACER (60%)

**Missão: "Casa Inteligente"**
Criar uma simulação de casa inteligente usando Packet Tracer.

1. **Infraestrutura Básica (20 pontos)**
   ```plaintext
   Configurar:
   - Roteador wireless
   - Switch
   - Gateway IoT
   - Servidor local
   ```

2. **Dispositivos IoT (20 pontos)**
   ```plaintext
   Implementar:
   - Sensores de movimento
   - Controle de iluminação
   - Termostatos
   - Câmeras de segurança
   ```

3. **Automação (20 pontos)**
   ```plaintext
   Criar regras para:
   - Economia de energia
   - Segurança
   - Conforto
   - Monitoramento
   ```

### ⭐ Pontos Extras (10 pontos)
- Dashboard de monitoramento
- Controle por smartphone
- Cenários de automação
- Integração com serviços web
